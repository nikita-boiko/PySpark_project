# Riiid AIEd Challenge: Exploratory Data Analysis with PySpark

Этот проект демонстрирует использование **PySpark** для обработки и анализа больших данных на примере соревнования [Riiid! Answer Correctness Prediction](https://www.kaggle.com).

## 📌 Описание
В репозитории представлен рабочий процесс (pipeline) по загрузке, очистке и первичному анализу данных об обучении студентов. Основной упор сделан на высокую производительность за счет использования **Apache Arrow** и распределенных вычислений Spark.

## 🛠 Стек технологий
- **KaggleHub API**: для программной загрузки датасетов.
- **PySpark (SQL, MLlib)**: для обработки данных и статистических вычислений.
- **Pandas API on Spark**: для удобного манипулирования данными в стиле Pandas.
- **Seaborn & Matplotlib**: для визуализации результатов.

## 🚀 Основные этапы работы
1. **Загрузка данных**: Использование `kagglehub` для автоматического скачивания файлов соревнования.
2. **Инициализация Spark**: Настройка `SparkSession` с оптимизацией `Arrow` для быстрого обмена данными.
3. **Очистка данных**: 
   - Изменение типов данных (кастинг признаков в `IntegerType`).
   - Анализ доли пропущенных значений.
   - Удаление строк с `null` значениями.
4. **Корреляционный анализ**:
   - Преобразование признаков в векторы через `VectorAssembler`.
   - Расчет матрицы корреляции Пирсона с помощью `Correlation.corr`.
5. **Визуализация**: Построение тепловой карты (heatmap) для поиска зависимостей между факторами обучения и успеваемостью.
